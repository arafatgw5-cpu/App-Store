---

## ✨ Overview

This is a modern **App Store Web Application** built using Next.js, where users can browse apps, view details, and install apps using local storage.

The project focuses on clean UI, smooth UX, and real-world app store simulation.

---

## 🚀 Features

- 📦 Browse all apps
- 🔍 View app details page
- 📥 Install apps (using localStorage)
- 📂 Installed apps page
- ⚡ Fast and responsive UI
- 🎨 Modern design using Tailwind CSS & DaisyUI

---

## 🛠️ Tech Stack

- ⚛️ Next.js (App Router)
- 🎨 Tailwind CSS
- 🌼 DaisyUI
- 📦 LocalStorage (for install system)
